# Интеграция Pushok Zigbee Hub для Home Assistant

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://github.com/hacs/integration)

Интеграция Home Assistant для Pushok Zigbee Hub.

## Возможности

- Поддержка локального и удалённого подключения
- Push-коммуникация через WebSocket
- Автоматическое обнаружение устройств
- Поддержка различных Zigbee устройств:
  - Выключатели и умные розетки
  - Сенсоры (температура, влажность, мощность и др.)
  - Бинарные сенсоры (движение, дверь/окно и др.)
  - Освещение с регулировкой яркости и цветовой температуры
  - Числовые элементы управления (слайдеры)
  - Элементы выбора (выпадающие списки)

## Установка

### HACS (Рекомендуется)

1. Откройте HACS в Home Assistant
2. Нажмите "Интеграции"
3. Нажмите меню с тремя точками и выберите "Пользовательские репозитории"
4. Добавьте URL репозитория: `https://github.com/kosyakow/ha-pushok-hub`
5. Выберите категорию: "Интеграция"
6. Нажмите "Добавить"
7. Найдите "Pushok Zigbee Hub" в списке и установите
8. Перезапустите Home Assistant

### Ручная установка

1. Скачайте папку `custom_components/pushok_hub`
2. Скопируйте её в директорию `config/custom_components/` вашего Home Assistant
3. Перезапустите Home Assistant

## Настройка

### Первичная настройка

1. Перейдите в Настройки → Устройства и службы
2. Нажмите "Добавить интеграцию"
3. Найдите "Pushok Zigbee Hub"
4. Введите IP-адрес и порт хаба (для первичной настройки хаб должен быть в режиме сопряжения)
5. Интеграция зарегистрируется на хабе и сохранит ключи аутентификации

### Переключение на удалённое подключение

После первичной локальной настройки можно переключиться на удалённое подключение (через облачный шлюз):

1. Перейдите в Настройки → Устройства и службы
2. Найдите "Pushok Zigbee Hub" и нажмите на него
3. Нажмите меню с тремя точками → "Перенастроить"
4. Выберите тип подключения "Remote (via cloud)"
5. Введите ID вашего хаба

Это позволяет получить доступ к хабу из любой точки без необходимости открывать его в интернет. Ключи аутентификации сохраняются при смене типа подключения.

## Поддерживаемые устройства

Интеграция автоматически обнаруживает устройства, подключённые к вашему Pushok Hub, и создаёт соответствующие сущности на основе возможностей устройств.
